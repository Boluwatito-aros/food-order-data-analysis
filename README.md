# Food Order Data Analysis

A data science case study performing exploratory data analysis on a food aggregator company's dataset.

## View the notebook
The full analysis is in `Boluwatito_PYF_Project_LearnerNotebook.ipynb` above.

## About
Analysed a food aggregator company's order dataset to uncover demand patterns across
registered customers and partner restaurants, directly informing customer experience strategy.

## What's inside
- End-to-end EDA using Pandas and NumPy — cleaning missing values, standardising data types,
  and engineering features such as order frequency per restaurant and average spend per customer
- Visualisations using Matplotlib and Seaborn — heatmaps, bar charts, and distribution plots
  highlighting peak ordering times, top-performing restaurants, and cuisine preferences
- Business insights — top 10 restaurants by order volume, repeat-customer behaviour,
  and underperforming cuisines
- Structured analytical report with findings and recommendations

## Tech
Python · Pandas · NumPy · Matplotlib · Seaborn · Jupyter Notebook · EDA
